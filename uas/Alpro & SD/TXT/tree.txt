Apa yang dimaksud dengan struktur data tree?
A. Kumpulan elemen dengan akses acak
B. Kumpulan elemen dengan prinsip LIFO
C. Kumpulan elemen yang tersusun secara hierarkis
D. Kumpulan elemen yang tidak tersusun
ANSWER: C

Apa yang membedakan binary tree dengan tree pada umumnya?
A. Binary tree hanya memiliki dua cabang untuk setiap node
B. Binary tree hanya memiliki satu cabang untuk setiap node
C. Binary tree tidak memiliki cabang
D. Binary tree memiliki banyak cabang untuk setiap node
ANSWER: A

Apa yang dimaksud dengan binary search tree (BST)?
A. Binary tree di mana semua node memiliki dua anak
B. Binary tree di mana setiap node memiliki nilai lebih besar dari anak kiri dan lebih kecil dari anak kanan
C. Binary tree di mana setiap node memiliki nilai sama
D. Binary tree di mana setiap node memiliki nilai yang tidak terbatas
ANSWER: B

Dalam binary search tree (BST), apa yang harus berlaku untuk setiap node?
A. Nilai node kiri harus lebih besar dari nilai node.
B. Nilai node kanan harus lebih besar dari nilai node.
C. Nilai node kiri harus lebih kecil dari nilai node, dan nilai node kanan harus lebih besar dari nilai node.
D. Nilai node kiri harus sama dengan nilai node.
ANSWER: C

Apa yang dimaksud dengan traversal inorder pada binary tree?
A. Mengunjungi node kiri, node kanan, kemudian node akar
B. Mengunjungi node akar, node kiri, kemudian node kanan
C. Mengunjungi node kiri, node akar, kemudian node kanan
D. Tidak ada traversal jenis ini dalam binary tree
ANSWER: C

Apa yang dimaksud dengan tinggi (height) dari sebuah binary tree?
A. Jumlah node dalam tree
B. Jumlah cabang dalam tree
C. Jumlah level dalam tree
D. Jumlah nilai dalam tree
ANSWER: C

Berapa banyak anak maksimum yang dapat dimiliki oleh setiap node dalam binary tree?
A. 0
B. 1
C. 2
D. 3
ANSWER: C

Apa yang dimaksud dengan node dalam konteks tree?
A. Sebuah elemen data yang menyimpan data dan alamat node sebelumnya.
B. Sebuah elemen data yang menyimpan data dan alamat node berikutnya.
C. Sebuah elemen data yang menyimpan data dan alamat anak-anaknya.
D. Sebuah elemen data yang hanya menyimpan data tanpa alamat node lain.
ANSWER: C

Jika sebuah BST berisi elemen 15, 10, 20, 8, 12, 17, 25, elemen apa yang akan berada di node kiri dari root?
A. 10
B. 8
C. 12
D. 17
ANSWER: A

Jika sebuah BST berisi elemen 15, 10, 20, 8, 12, 17, 25, elemen apa yang akan menjadi root?
A. 10
B. 8
C. 15
D. 17
ANSWER: C

Dalam sebuah BST, bagaimana cara menambahkan elemen baru?
A. Mulai dari root, bandingkan elemen baru dengan node saat ini dan pindah ke anak kiri jika lebih kecil atau ke anak kanan jika lebih besar, ulangi hingga menemukan posisi yang sesuai.
B. Mulai dari node terakhir, bandingkan elemen baru dengan node saat ini dan pindah ke anak kiri jika lebih besar atau ke anak kanan jika lebih kecil, ulangi hingga menemukan posisi yang sesuai.
C. Tambahkan elemen baru ke anak kiri dari root jika lebih besar atau ke anak kanan jika lebih kecil.
D. Tambahkan elemen baru ke anak kanan dari node terakhir yang ditambahkan.
ANSWER: A

Dalam sebuah BT yang diimplementasikan menggunakan linked list, bagaimana cara mencari elemen tertentu?
A. Mulai dari node terakhir dan pindah ke node sebelumnya hingga menemukan elemen yang sesuai.
B. Mulai dari root dan lakukan pencarian traversal preorder, inorder, atau postorder untuk menemukan elemen yang sesuai.
C. Mulai dari root dan pindah ke node anak kiri jika elemen lebih kecil atau ke anak kanan jika lebih besar.
D. Mulai dari node yang memiliki elemen terkecil dan pindah ke node yang memiliki elemen terbesar.
ANSWER: B

Jika sebuah BT diimplementasikan menggunakan linked list, apa yang harus dilakukan untuk menghitung jumlah total node?
A. Menggunakan traversal preorder, inorder, atau postorder untuk menghitung node satu per satu.
B. Menggunakan traversal level-order dan menghitung node satu per satu.
C. Menambahkan penunjuk tambahan ke setiap node untuk menyimpan jumlah total node.
D. Menggunakan pencarian binary untuk menemukan semua node.
ANSWER: A
