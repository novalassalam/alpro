Berikut berhubungan dengan linked list, kecuali?
A. Pointer
B. Sekuensial
C. Node
D. Array
ANSWER: D

Kelebihan linked list menggunakan dibanding array adalah?
A. linked list tak terbatas (Dinamis)
B. Array lebih Dinamis
C. linked list terbatas
D. array unlimited
ANSWER: A

Apa perbedaan utama antara single linked list dan double linked list?
A. Single linked list memiliki alamat next, sedangkan double linked list memiliki alamat prev dan next.
B. Single linked list memiliki alamat prev, sedangkan double linked list memiliki alamat next.
C. Single linked list memiliki dua alamat next, sedangkan double linked list hanya memiliki satu alamat next.
D. Single linked list memiliki alamat next dan prev, sedangkan double linked list hanya memiliki alamat next.
ANSWER: A

Apa yang terjadi ketika node dihapus dari single linked list?
A. Node berikutnya akan kehilangan alamat prev.
B. Node sebelumnya akan kehilangan alamat next.
C. Semua node akan dihapus dari linked list.
D. Data dalam node dihapus, tetapi alamat next tetap ada.
ANSWER: B

Berikut merupakan komponen dalam single linked list ?
A. data dan alamat next
B. alamat next dan data
C. alamat prev, data, dan alamt next
D. null dan data
ANSWER: A

Dalam linked list operator yang digunakan menambah element di akhir element adalah?
A. Insert Last
B. Insert First
C. Insert After
D. Insert Before
ANSWER: A

Langkah terakhir dalam melakukan operator insert last pada SLL adalah
A. Siapkan element yang ada
B. Buat node kosong
C. Isi node kosong dengan value
D. Tambahkan ke akhir list yang sudah ada
ANSWER: D

Apa itu double linked list?
A. Sebuah daftar yang hanya dapat diakses dari depan ke belakang
B. Sebuah daftar yang hanya dapat diakses dari belakang ke depan
C. Sebuah daftar yang dapat diakses dari kedua arah
D. Sebuah daftar yang hanya dapat diakses dari tengah ke ujung
ANSWER: C

Bagaimana cara menambahkan node baru di awal double linked list?
A. Menetapkan NULL pada pointer node baru
B. Mengubah pointer head menjadi node baru
C. Mengubah pointer tail menjadi node baru
D. Menyisipkan node baru di antara dua node yang sudah ada
ANSWER: B

Berapa banyak pointer yang dimiliki oleh setiap node pada double linked list?
A. Satu
B. Dua
C. Tiga
D. Empat
ANSWER: B

Apa keuntungan utama menggunakan double linked list dibandingkan single linked list?
A. Kemampuan untuk menelusuri daftar dari kedua arah.
B. Penggunaan memori yang lebih efisien.
C. Implementasi lebih sederhana.
D. Lebih sedikit kebutuhan untuk memperbarui pointer saat menghapus node.
ANSWER: A

Dalam double linked list, apa yang terjadi ketika sebuah node baru ditambahkan di tengah-tengah daftar?
A. Node baru hanya menyesuaikan alamat next dari node sebelumnya.
B. Node baru hanya menyesuaikan alamat prev dari node berikutnya.
C. Node baru menyesuaikan alamat prev dari node berikutnya dan alamat next dari node sebelumnya.
D. Node baru hanya menyesuaikan data dan tidak ada perubahan pada alamat next atau prev.
ANSWER: C

Bagaimana kita dapat menentukan apakah double linked list kosong?
A. Memeriksa apakah alamat next dari node pertama adalah null.
B. Memeriksa apakah alamat prev dari node terakhir adalah null.
C. Memeriksa apakah alamat next dari node terakhir adalah null.
D. Memeriksa apakah head atau tail adalah null.
ANSWER: D

Apa yang terjadi jika kita mencoba menelusuri double linked list dari node terakhir ke node pertama?
A. Menggunakan alamat next dari setiap node.
B. Menggunakan alamat prev dari setiap node.
C. Menggunakan data dari setiap node.
D. Menggunakan kedua alamat next dan prev dari setiap node.
ANSWER: B

Dalam operasi penghapusan pada double linked list, jika kita menghapus node dengan data tertentu, bagaimana cara kita memastikan bahwa seluruh node dengan data yang sama dihapus?
A. Menelusuri daftar dari awal hingga akhir, menghapus setiap node dengan data yang sesuai, dan memperbarui pointer prev dan next.
B. Menggunakan algoritma rekursif untuk menghapus node yang sesuai.
C. Menggunakan fungsi hash untuk menemukan semua node yang memiliki data yang sama.
D. Mengubah struktur data untuk menggunakan daftar berantai.
ANSWER: A

Jika kita ingin membalik urutan elemen dalam double linked list, apa langkah pertama yang harus dilakukan?
A. Menukar data setiap node dengan node yang berlawanan.
B. Menukar alamat prev dan next dari setiap node.
C. Menukar head dan tail dari linked list.
D. Menambahkan node baru di akhir linked list.
ANSWER: B

Bagaimana cara menghapus node terakhir dalam single linked list?
A. Menelusuri dari awal hingga menemukan node terakhir, lalu menghapusnya.
B. Menelusuri dari awal hingga node sebelum terakhir, lalu mengatur alamat next dari node ini menjadi null.
C. Menggunakan algoritma rekursif untuk menemukan node terakhir dan menghapusnya.
D. Menukar data node terakhir dengan node sebelumnya, lalu menghapus node terakhir.
ANSWER: B

Bagaimana cara menghapus node tengah dalam single linked list jika hanya diberikan pointer ke node tersebut?
A. Menyalin data dari node berikutnya ke node yang akan dihapus, lalu menghapus node berikutnya.
B. Menyalin data dari node sebelumnya ke node yang akan dihapus, lalu menghapus node sebelumnya.
C. Mengatur alamat next dari node yang akan dihapus menjadi null.
D. Menghapus seluruh linked list dan membuat ulang tanpa node tersebut.
ANSWER: A
